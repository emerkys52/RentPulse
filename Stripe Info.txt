1. Create a Stripe Account (if you haven't)

  Go to https://dashboard.stripe.com and sign up or log in.

  2. Create a Product and Price

  1. Go to Products ? Add product
  2. Name: RentalPulse Premium
  3. Click Add product
  4. Under Pricing, click Add price:
    - Price: $4.99
    - Billing period: Monthly
    - Click Add price
  5. Copy the Price ID (starts with price_)

  3. Set Up Webhook

  1. Go to Developers ? Webhooks
  2. Click Add endpoint
  3. Endpoint URL: https://rentpulse-production.up.railway.app/api/webhooks/stripe
  4. Select events to listen to:
    - checkout.session.completed
    - customer.subscription.updated
    - customer.subscription.deleted
    - invoice.payment_failed
  5. Click Add endpoint
  6. Copy the Signing secret (starts with whsec_)

  4. Get API Keys

  1. Go to Developers ? API keys
  2. Copy your Secret key (starts with sk_live_ or sk_test_)

  5. Add Environment Variables to Railway

  Run these commands or add them via the Railway dashboard:

  railway link --project 61b9b1c2-c2f8-4c1a-895d-a845a46a0a27 --service RentalPulse
  railway variables set STRIPE_SECRET_KEY=sk_live_your_key_here
  railway variables set STRIPE_PRICE_ID=price_your_price_id_here
  railway variables set STRIPE_WEBHOOK_SECRET=whsec_your_webhook_secret_here1. Create a Stripe Account (if you haven't)
